Video Game Project. Mutliplayer Tank game.
P1:
W - Forwards.
S - Backwards.
A D - left and right turn.
Space - shoot.

P2:
Up Arrow - Forwards.
Down Arrow - Backwards.
Left and Right - left and right turn.
Shift - shoot.
